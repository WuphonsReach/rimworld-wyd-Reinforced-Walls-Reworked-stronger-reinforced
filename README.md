# rimworld-wyd-Reinforced-Walls-Reworked-stronger-reinforced
The base fortified wall in "[WYD] Reinforced Walls Reworked" has a good health amount, but the reinforced versions aren't strong enough for my tastes.  This kicks the HP of the reinforced wall from 1200 to 1600 and the embrasure from 900 to 1400.
